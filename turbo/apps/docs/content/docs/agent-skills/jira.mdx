[Jira](https://www.atlassian.com/software/jira) is Atlassian's project management tool for tracking issues, sprints, and workflows.

## Required Environment

| Name             | Type   | Description                                                                                     |
| ---------------- | ------ | ----------------------------------------------------------------------------------------------- |
| `JIRA_API_TOKEN` | secret | API token from [Atlassian Account](https://id.atlassian.com/manage-profile/security/api-tokens) |
| `JIRA_DOMAIN`    | var    | Your Jira domain (e.g., `mycompany.atlassian.net`)                                              |
| `JIRA_EMAIL`     | var    | Email associated with your Atlassian account                                                    |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/jira # [!code highlight]
```

## Run

```bash
vm0 run my-agent "create a bug ticket in Jira" \
  --secrets JIRA_API_TOKEN=xxx --vars JIRA_DOMAIN=xxx --vars JIRA_EMAIL=xxx # [!code highlight]
```
