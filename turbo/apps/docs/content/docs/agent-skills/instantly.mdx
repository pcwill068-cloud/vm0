[Instantly](https://instantly.ai/) is a cold email outreach platform for managing campaigns, leads, and email accounts.

## Required Environment

| Name                | Type   | Description                                                                           |
| ------------------- | ------ | ------------------------------------------------------------------------------------- |
| `INSTANTLY_API_KEY` | secret | API key from [Instantly Settings](https://app.instantly.ai/app/settings/integrations) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/instantly # [!code highlight]
```

## Run

```bash
vm0 run my-agent "create email campaign" \
  --secrets INSTANTLY_API_KEY=xxx # [!code highlight]
```
