[Plausible](https://plausible.io/) is a privacy-friendly web analytics platform for tracking visitors, pageviews, and traffic sources.

## Required Environment

| Name                | Type   | Description                                        |
| ------------------- | ------ | -------------------------------------------------- |
| `PLAUSIBLE_API_KEY` | secret | API key from Plausible Account Settings > API Keys |
| `PLAUSIBLE_SITE_ID` | var    | Your site domain (e.g., `example.com`)             |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/plausible # [!code highlight]
```

## Run

```bash
vm0 run my-agent "get website analytics" \
  --secrets PLAUSIBLE_API_KEY=xxx --vars PLAUSIBLE_SITE_ID=xxx # [!code highlight]
```
