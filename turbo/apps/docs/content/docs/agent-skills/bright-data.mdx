---

[Bright Data](https://brightdata.com/) provides web scraping infrastructure with residential proxies and pre-built data collectors. This skill enables your agent to scrape websites at scale.

## Required Environment

| Name                 | Type   | Description                                                                   |
| -------------------- | ------ | ----------------------------------------------------------------------------- |
| `BRIGHTDATA_API_KEY` | secret | API key from [Bright Data Dashboard](https://brightdata.com/cp/setting/users) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/bright-data # [!code highlight]
```

## Run

```bash
vm0 run my-agent "scrape social media data" \
  --secrets BRIGHTDATA_API_KEY=xxx # [!code highlight]
```
