[Figma](https://www.figma.com/) is a collaborative design platform. Use the API to access design files, export images, and manage comments.

## Required Environment

| Name              | Type   | Description                                                                 |
| ----------------- | ------ | --------------------------------------------------------------------------- |
| `FIGMA_API_TOKEN` | secret | Personal access token from [Figma Settings](https://www.figma.com/settings) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/figma # [!code highlight]
```

## Run

```bash
vm0 run my-agent "export frames from Figma file" \
  --secrets FIGMA_API_TOKEN=figd_xxx # [!code highlight]
```

```markdown title="AGENTS.md"
# Design Export Agent

You export assets from Figma design files.

1. Receive Figma file key
2. Get file structure and frames

## Export Formats

- PNG, JPG, SVG, PDF
- Scale: 1x, 2x, 4x
```
