---
title: SerpApi
description: Search engine results scraping API
---

[SerpApi](https://serpapi.com/) scrapes search engine results from Google, Bing, YouTube, and more for SEO analysis and research.

## Required Environment

| Name              | Type   | Description                                            |
| ----------------- | ------ | ------------------------------------------------------ |
| `SERPAPI_API_KEY` | secret | API key from [SerpApi Dashboard](https://serpapi.com/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/serpapi # [!code highlight]
```

## Run

```bash
vm0 run my-agent "get Google search results" \
  --secrets SERPAPI_API_KEY=xxx # [!code highlight]
```
