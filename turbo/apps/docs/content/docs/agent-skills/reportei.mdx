[Reportei](https://www.reportei.com/) is a marketing report generation platform with automated analytics for managing clients, reports, and integrations.

## Required Environment

| Name                 | Type   | Description                       |
| -------------------- | ------ | --------------------------------- |
| `REPORTEI_API_TOKEN` | secret | API token from Reportei Dashboard |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/reportei # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set REPORTEI_API_TOKEN your-reportei-api-token
```

Then run your agent - secret is automatically loaded:

```bash
vm0 run my-agent "generate marketing report"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets REPORTEI_API_TOKEN=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
