---
title: RSS Fetch
description: Fetch and parse RSS/Atom feeds
---

RSS Feed Fetcher allows fetching and parsing RSS/Atom feeds from any source to gather news and articles.

## Required Environment

No API key required. RSS feeds are public.

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/rss-fetch # [!code highlight]
```

## Run

```bash
vm0 run my-agent "fetch latest articles from RSS feed"
```
