---
title: PDF.co
description: PDF processing and extraction API
---

[PDF.co](https://pdf.co/) provides PDF processing including extraction, conversion, merging, splitting, and OCR with AI parsing.

## Required Environment

| Name            | Type   | Description                                    |
| --------------- | ------ | ---------------------------------------------- |
| `PDFCO_API_KEY` | secret | API key from [PDF.co App](https://app.pdf.co/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/pdfco # [!code highlight]
```

## Run

```bash
vm0 run my-agent "extract text from PDF" \
  --secrets PDFCO_API_KEY=xxx # [!code highlight]
```
