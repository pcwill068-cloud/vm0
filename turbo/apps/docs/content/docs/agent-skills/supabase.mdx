[Supabase](https://supabase.com/) is an open-source Firebase alternative providing PostgreSQL database with auto-generated REST API.

## Required Environment

| Name                       | Type   | Description                                        |
| -------------------------- | ------ | -------------------------------------------------- |
| `SUPABASE_SECRET_KEY`      | secret | Service role key (secret) from Settings > API Keys |
| `SUPABASE_URL`             | var    | Project URL from Supabase Dashboard                |
| `SUPABASE_PUBLISHABLE_KEY` | var    | Anon key (public) from Settings > API Keys         |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/supabase # [!code highlight]
```

## Run

```bash
vm0 run my-agent "query database" \
  --secrets SUPABASE_SECRET_KEY=xxx --vars SUPABASE_URL=xxx SUPABASE_PUBLISHABLE_KEY=xxx # [!code highlight]
```
