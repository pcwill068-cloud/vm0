[Intercom](https://www.intercom.com/) is a customer messaging platform for managing conversations, contacts, and support operations.

## Required Environment

| Name                    | Type   | Description                              |
| ----------------------- | ------ | ---------------------------------------- |
| `INTERCOM_ACCESS_TOKEN` | secret | Access token from Intercom Developer Hub |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/intercom # [!code highlight]
```

## Run

```bash
vm0 run my-agent "list open conversations" \
  --secrets INTERCOM_ACCESS_TOKEN=xxx # [!code highlight]
```

## Example Instructions

```markdown title="AGENTS.md"
# Support Agent

You manage Intercom conversations.

## Workflow

2. Read conversation messages

## Conversation States

- open, closed, snoozed
```
