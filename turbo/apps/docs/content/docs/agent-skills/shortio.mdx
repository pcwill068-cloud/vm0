---

[Short.io](https://short.io/) creates and manages short links on branded domains with click tracking and analytics.

## Required Environment

| Name                | Type   | Description                                                                          |
| ------------------- | ------ | ------------------------------------------------------------------------------------ |
| `SHORTIO_API_KEY`   | secret | API key from [Short.io Settings](https://app.short.io/settings/integrations/api-key) |
| `SHORTIO_DOMAIN`    | var    | Your branded domain (e.g., `link.example.com`)                                       |
| `SHORTIO_DOMAIN_ID` | var    | Domain ID (needed for list and statistics operations, optional)                      |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/shortio # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set SHORTIO_API_KEY your-shortio-api-key
```

Then run your agent - secret is automatically loaded:

```bash
vm0 run my-agent "create short link"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets SHORTIO_API_KEY=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
