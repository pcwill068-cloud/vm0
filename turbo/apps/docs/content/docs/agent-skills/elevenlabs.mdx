[ElevenLabs](https://elevenlabs.io/) provides realistic AI voice generation and text-to-speech with natural-sounding voices.

## Required Environment

| Name                 | Type   | Description                                                                    |
| -------------------- | ------ | ------------------------------------------------------------------------------ |
| `ELEVENLABS_API_KEY` | secret | API key from [ElevenLabs Profile](https://elevenlabs.io/app/settings/api-keys) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/elevenlabs # [!code highlight]
```

## Run

```bash
vm0 run my-agent "convert text to speech" \
  --secrets ELEVENLABS_API_KEY=xxx # [!code highlight]
```
