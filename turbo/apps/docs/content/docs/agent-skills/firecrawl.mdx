[Firecrawl](https://www.firecrawl.dev/) is a web scraping API that helps you extract clean, structured data from any website.

## Required Environment

| Name                | Type   | Description                                                    |
| ------------------- | ------ | -------------------------------------------------------------- |
| `FIRECRAWL_API_KEY` | secret | API key from [Firecrawl Dashboard](https://www.firecrawl.dev/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/firecrawl # [!code highlight]
```

## Run

```bash
vm0 run my-agent "scrape the website" \
  --secrets FIRECRAWL_API_KEY=fc-xxx # [!code highlight]
```
