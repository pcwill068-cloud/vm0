---

[HTMLCSStoImage](https://htmlcsstoimage.com/) generates images from HTML/CSS or captures screenshots of web pages for social cards, thumbnails, and certificates.

## Required Environment

| Name           | Type   | Description                                                         |
| -------------- | ------ | ------------------------------------------------------------------- |
| `HCTI_API_KEY` | secret | API key from [HCTI Dashboard](https://htmlcsstoimage.com/dashboard) |
| `HCTI_USER_ID` | var    | User ID from HCTI Dashboard                                         |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/htmlcsstoimage # [!code highlight]
```

## Run

```bash
vm0 run my-agent "generate OG image from HTML" \
  --secrets HCTI_API_KEY=xxx --vars HCTI_USER_ID=xxx # [!code highlight]
```
