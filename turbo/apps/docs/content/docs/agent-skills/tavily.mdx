[Tavily](https://tavily.com/) is an AI search API designed for retrieval-augmented generation (RAG) with real-time web search and source citations.

## Required Environment

| Name             | Type   | Description                                          |
| ---------------- | ------ | ---------------------------------------------------- |
| `TAVILY_API_KEY` | secret | API key from [Tavily Dashboard](https://tavily.com/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/tavily # [!code highlight]
```

## Run

```bash
vm0 run my-agent "search for latest news" \
  --secrets TAVILY_API_KEY=tvly-xxx # [!code highlight]
```
