title: Pushinator
description: Push notification API for mobile devices
---

[Pushinator](https://pushinator.com/) sends push notifications to mobile devices for alerts, deployments, and task completion notifications.

## Required Environment

| Name                 | Type   | Description                                                                |
| -------------------- | ------ | -------------------------------------------------------------------------- |
| `PUSHINATOR_API_KEY` | secret | API token from [Pushinator Console](https://console.pushinator.com/tokens) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/pushinator # [!code highlight]
```

## Run

```bash
vm0 run my-agent "send push notification" \
  --secrets PUSHINATOR_API_KEY=xxx # [!code highlight]
```
