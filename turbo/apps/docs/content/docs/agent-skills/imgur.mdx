[Imgur](https://imgur.com/) is a free image hosting service for uploading images and getting shareable URLs.

## Required Environment

| Name              | Type   | Description                                                        |
| ----------------- | ------ | ------------------------------------------------------------------ |
| `IMGUR_CLIENT_ID` | secret | Client ID from [Imgur API](https://api.imgur.com/oauth2/addclient) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/imgur # [!code highlight]
```

## Run

```bash
vm0 run my-agent "upload image to Imgur" \
  --secrets IMGUR_CLIENT_ID=xxx # [!code highlight]
```
