[Streak](https://www.streak.com/) is a CRM built inside Gmail for managing pipelines, deals, contacts, and email threads.

## Required Environment

| Name             | Type   | Description                                             |
| ---------------- | ------ | ------------------------------------------------------- |
| `STREAK_API_KEY` | secret | API key from Gmail > Streak > Integrations > Streak API |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/streak # [!code highlight]
```

## Run

```bash
vm0 run my-agent "create deal in Streak" \
  --secrets STREAK_API_KEY=xxx # [!code highlight]
```
