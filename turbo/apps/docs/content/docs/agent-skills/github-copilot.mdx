title: GitHub Copilot
description: Manage Copilot subscriptions and usage metrics
---

[GitHub Copilot](https://github.com/features/copilot) API allows managing Copilot subscriptions and retrieving usage metrics for organizations.

## Required Environment

| Name           | Type   | Description                                               |
| -------------- | ------ | --------------------------------------------------------- |
| `GITHUB_TOKEN` | secret | Personal access token with `manage_billing:copilot` scope |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/github-copilot # [!code highlight]
```

## Run

```bash
vm0 run my-agent "get Copilot usage metrics" \
  --secrets GITHUB_TOKEN=ghp_xxx # [!code highlight]
```
