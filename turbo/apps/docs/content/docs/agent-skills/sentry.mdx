[Sentry](https://sentry.io/) is an error tracking platform for managing issues, projects, and releases.

## Required Environment

| Name           | Type   | Description                                                       |
| -------------- | ------ | ----------------------------------------------------------------- |
| `SENTRY_TOKEN` | secret | Auth token from Sentry Developer Settings > Internal Integrations |
| `SENTRY_HOST`  | var    | Sentry host (e.g., `sentry.io` or self-hosted)                    |
| `SENTRY_ORG`   | var    | Your Sentry organization slug                                     |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/sentry # [!code highlight]
```

## Run

```bash
vm0 run my-agent "list recent errors" \
  --secrets SENTRY_TOKEN=sntrys_xxx --vars SENTRY_HOST=xxx SENTRY_ORG=xxx # [!code highlight]
```
