[Lark](https://www.larksuite.com/) (also known as Feishu) is an enterprise collaboration platform for messaging, group management, contacts, and calendar.

## Required Environment

| Name              | Type   | Description                                                        |
| ----------------- | ------ | ------------------------------------------------------------------ |
| `LARK_APP_SECRET` | secret | App secret from [Lark Open Platform](https://open.larkoffice.com/) |
| `LARK_APP_ID`     | var    | App ID from Lark Open Platform                                     |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/lark # [!code highlight]
```

## Run

```bash
vm0 run my-agent "send message to Lark group" \
  --secrets LARK_APP_SECRET=xxx --vars LARK_APP_ID=xxx # [!code highlight]
```
