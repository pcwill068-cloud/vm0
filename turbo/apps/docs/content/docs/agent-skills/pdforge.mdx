---
title: PDForge
description: PDF generation from templates and HTML
---

[PDForge](https://pdfnoodle.com/) (formerly PDF Noodle) generates PDFs from reusable templates or raw HTML for invoices, reports, and documents.

## Required Environment

| Name              | Type   | Description                                                 |
| ----------------- | ------ | ----------------------------------------------------------- |
| `PDFORGE_API_KEY` | secret | API key from [PDF Noodle Dashboard](https://pdfnoodle.com/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/pdforge # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set PDFORGE_API_KEY your-pdforge-api-key
```

Then run your agent - secret is automatically loaded:

```bash
vm0 run my-agent "generate PDF from template"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets PDFORGE_API_KEY=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
