---

[ZapSign](https://www.zapsign.com.br/) provides electronic signatures with legal validity for documents, contracts, and forms.

## Required Environment

| Name                | Type   | Description                                                  |
| ------------------- | ------ | ------------------------------------------------------------ |
| `ZAPSIGN_API_TOKEN` | secret | API token from ZapSign Settings > Integrations > ZAPSIGN API |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/zapsign # [!code highlight]
```

## Run

```bash
vm0 run my-agent "create document for signature" \
  --secrets ZAPSIGN_API_TOKEN=xxx # [!code highlight]
```

## Example Instructions

```markdown title="AGENTS.md"

- Email, SMS, selfie verification
```
