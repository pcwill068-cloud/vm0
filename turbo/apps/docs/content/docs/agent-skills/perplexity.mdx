[Perplexity](https://perplexity.ai/) is an AI search engine that provides real-time web-grounded answers with source citations.

## Required Environment

| Name                 | Type   | Description                                                           |
| -------------------- | ------ | --------------------------------------------------------------------- |
| `PERPLEXITY_API_KEY` | secret | API key from [Perplexity Settings](https://perplexity.ai/account/api) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/perplexity # [!code highlight]
```

## Run

```bash
vm0 run my-agent "search for latest tech news" \
  --secrets PERPLEXITY_API_KEY=pplx-xxx # [!code highlight]
```
