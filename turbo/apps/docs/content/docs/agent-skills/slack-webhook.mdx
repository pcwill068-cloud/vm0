---
title: Slack Webhook
description: Send messages to Slack via Incoming Webhooks
---

[Slack Incoming Webhooks](https://api.slack.com/messaging/webhooks) send messages to Slack channels without OAuth or bot setup.

## Required Environment

| Name                | Type   | Description                                    |
| ------------------- | ------ | ---------------------------------------------- |
| `SLACK_WEBHOOK_URL` | secret | Webhook URL from Slack App > Incoming Webhooks |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/slack-webhook # [!code highlight]
```

## Run

```bash
vm0 run my-agent "send status update to Slack" \
  --secrets SLACK_WEBHOOK_URL=https://hooks.slack.com/services/xxx # [!code highlight]
```
