[Qdrant](https://qdrant.tech/) is a vector database for storing and searching vector embeddings for RAG, semantic search, and recommendations.

## Required Environment

| Name             | Type   | Description                                                   |
| ---------------- | ------ | ------------------------------------------------------------- |
| `QDRANT_API_KEY` | secret | API key from [Qdrant Cloud](https://cloud.qdrant.io/)         |
| `QDRANT_URL`     | var    | Qdrant cluster URL (e.g., `https://xxx.cloud.qdrant.io:6333`) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/qdrant # [!code highlight]
```

## Run

```bash
vm0 run my-agent "search similar documents" \
  --secrets QDRANT_API_KEY=xxx --vars QDRANT_URL=xxx # [!code highlight]
```
