[Podchaser](https://www.podchaser.com/) provides comprehensive podcast data including shows, episodes, creators, charts, and sponsorship information.

## Required Environment

| Name                      | Type   | Description                               |
| ------------------------- | ------ | ----------------------------------------- |
| `PODCHASER_CLIENT_SECRET` | secret | Client secret from Podchaser API Settings |
| `PODCHASER_CLIENT_ID`     | var    | Client ID from Podchaser API Settings     |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/podchaser # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set PODCHASER_CLIENT_SECRET your-podchaser-client-secret
```

Then run your agent - secret is automatically loaded:

```bash
vm0 run my-agent "search podcasts about technology"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets PODCHASER_CLIENT_SECRET=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
