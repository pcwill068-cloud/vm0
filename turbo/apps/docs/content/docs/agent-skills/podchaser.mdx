[Podchaser](https://www.podchaser.com/) provides comprehensive podcast data including shows, episodes, creators, charts, and sponsorship information.

## Required Environment

| Name                      | Type   | Description                               |
| ------------------------- | ------ | ----------------------------------------- |
| `PODCHASER_CLIENT_SECRET` | secret | Client secret from Podchaser API Settings |
| `PODCHASER_CLIENT_ID`     | var    | Client ID from Podchaser API Settings     |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/podchaser # [!code highlight]
```

## Run

```bash
vm0 run my-agent "search podcasts about technology" \
  --secrets PODCHASER_CLIENT_SECRET=xxx --vars PODCHASER_CLIENT_ID=xxx # [!code highlight]
```
