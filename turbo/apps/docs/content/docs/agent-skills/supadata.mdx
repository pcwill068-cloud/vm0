title: Supadata
description: Video transcript extraction and web scraping
---

[Supadata](https://supadata.ai/) extracts video transcripts from YouTube, TikTok, Instagram, and scrapes web content to markdown for AI processing.

## Required Environment

| Name               | Type   | Description                                                  |
| ------------------ | ------ | ------------------------------------------------------------ |
| `SUPADATA_API_KEY` | secret | API key from [Supadata Dashboard](https://dash.supadata.ai/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/supadata # [!code highlight]
```

## Run

```bash
vm0 run my-agent "extract YouTube transcript" \
  --secrets SUPADATA_API_KEY=xxx # [!code highlight]
```
