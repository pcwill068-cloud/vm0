---
title: PDF4ME
description: Comprehensive PDF processing API
---

[PDF4ME](https://pdf4me.com/) provides 60+ PDF operations including conversion, merge, split, compress, OCR, watermark, and digital signatures.

## Required Environment

| Name             | Type   | Description                                                                   |
| ---------------- | ------ | ----------------------------------------------------------------------------- |
| `PDF4ME_API_KEY` | secret | API key from [PDF4ME Dashboard](https://dev.pdf4me.com/dashboard/#/api-keys/) |

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/pdf4me # [!code highlight]
```

## Run

```bash
vm0 run my-agent "merge PDF files" \
  --secrets PDF4ME_API_KEY=xxx # [!code highlight]
```
