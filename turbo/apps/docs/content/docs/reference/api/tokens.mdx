Manage API tokens for authenticating with the VM0 API.

## List Tokens

```
GET /v1/tokens
```

Returns a paginated list of your API tokens.

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | integer | 20 | Items per page (1-100) |
| `cursor` | string | - | Pagination cursor |

### Request

```bash
curl https://www.vm0.ai/v1/tokens \
  -H "Authorization: Bearer vm0_live_xxx"
```

### Response

```json
{
  "data": [
    {
      "id": "tok_abc123",
      "name": "my-token",
      "token_prefix": "vm0_live_a1b2...",
      "last_used_at": "2024-01-15T10:30:00Z",
      "expires_at": "2024-04-15T00:00:00Z",
      "created_at": "2024-01-15T00:00:00Z"
    }
  ],
  "pagination": {
    "has_more": false,
    "next_cursor": null
  }
}
```

## Create Token

```
POST /v1/tokens
```

Creates a new API token.

<Callout type="warning">
  The full token value is only returned once during creation. Store it securely.
</Callout>

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Token name (1-100 characters) |
| `expires_in_days` | integer | No | Days until expiration (1-365, default: 90) |

### Request

```bash
curl https://www.vm0.ai/v1/tokens \
  -H "Authorization: Bearer vm0_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "ci-token",
    "expires_in_days": 30
  }'
```

### Response

```json
{
  "id": "tok_def456",
  "name": "ci-token",
  "token_prefix": "vm0_live_x9y8...",
  "token": "vm0_live_x9y8z7w6v5u4t3s2r1q0p9o8n7m6l5k4j3i2h1g0f9e8d7c6b5a4",
  "last_used_at": null,
  "expires_at": "2024-02-14T00:00:00Z",
  "created_at": "2024-01-15T00:00:00Z"
}
```

## Get Token

```
GET /v1/tokens/:id
```

Retrieves details of a specific token.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Token ID |

### Request

```bash
curl https://www.vm0.ai/v1/tokens/tok_abc123 \
  -H "Authorization: Bearer vm0_live_xxx"
```

### Response

```json
{
  "id": "tok_abc123",
  "name": "my-token",
  "token_prefix": "vm0_live_a1b2...",
  "last_used_at": "2024-01-15T10:30:00Z",
  "expires_at": "2024-04-15T00:00:00Z",
  "created_at": "2024-01-15T00:00:00Z"
}
```

## Delete Token

```
DELETE /v1/tokens/:id
```

Permanently revokes a token. This action cannot be undone.

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Token ID |

### Request

```bash
curl -X DELETE https://www.vm0.ai/v1/tokens/tok_abc123 \
  -H "Authorization: Bearer vm0_live_xxx"
```

### Response

Returns `204 No Content` on success.

---

## Token Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique token identifier |
| `name` | string | Human-readable name |
| `token_prefix` | string | First 16 characters of the token |
| `token` | string | Full token value (only on creation) |
| `last_used_at` | string | ISO 8601 timestamp of last use |
| `expires_at` | string | ISO 8601 expiration timestamp |
| `created_at` | string | ISO 8601 creation timestamp |
